# Nogada Tip Flow Application

A multi-portal system for managing car guard tipping and payments.

## Getting Started

### Prerequisites

The only requirement is having Node.js & npm installed.

### Installation

Follow these steps:

```sh
# Step 1: Navigate to the project directory.
cd nogada-tip-flow

# Step 2: Install the necessary dependencies.
npm install

# Step 3: Start the development server with auto-reloading and an instant preview.
npm run dev
```

The application will be available at http://localhost:8080

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## Application Structure

The application consists of three main portals:

1. **Car Guard App** - Mobile interface for car guards
2. **Customer Portal** - Customer tipping interface  
3. **Admin Application** - System administration dashboard
